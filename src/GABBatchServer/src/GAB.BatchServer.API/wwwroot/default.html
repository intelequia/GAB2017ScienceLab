GAB Science Lab Batch Server

API to pull task batches and push results to the GAB Science Lab

Created by David Rodriguez (@davidjrh), Adonai Suárez (@adonaisg) and Martin Abbott (@martinabbott), for the Global Azure Bootcamp 2017 Science Lab running Sebastian Hidalgo's Star Formation History SELIGA algorithm at the Instituo de Astrofisica de Canarias

See more at http://global.azurebootcamp.net
